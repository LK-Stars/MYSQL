# 💾 Projetos MySQL

Repositório dedicado aos meus estudos e práticas com **banco de dados MySQL**, incluindo modelagem, criação de schemas, consultas SQL e relacionamentos entre tabelas.

---

## 📌 Sobre o Projeto

Este repositório contém exercícios e projetos desenvolvidos durante minha graduação em **Análise e Desenvolvimento de Sistemas**, com foco em:

* Modelagem de banco de dados
* Criação de tabelas (DDL)
* Manipulação de dados (DML)
* Consultas SQL (SELECT, JOIN, GROUP BY, etc.)
* Relacionamentos entre entidades

---

## 🗂️ Estrutura do Repositório

📁 **db_clinica**

* Scripts de criação e manipulação de banco para uma clínica
* Tabelas como pacientes, exames, estados e regiões
* Uso de JOINs e relacionamentos

📁 **db_rh**

* Banco de dados voltado para Recursos Humanos
* Estrutura com funcionários, cargos e organização
* Exercícios de consultas e agregações

📁 **Diagramas (.mwb)**

* Modelagem visual feita no MySQL Workbench
* Representação das tabelas e seus relacionamentos

---
